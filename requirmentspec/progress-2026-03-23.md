### 功能範圍

| 端點 | 說明 |
|------|------|
| `POST /api/auth/register` | 以 email + username + password 註冊（重複 email → 409） |
| `POST /api/auth/login` | 登入取得 JWT Access Token（15 分鐘）+ Refresh Token（7 天） |
| `POST /api/auth/refresh` | 以 Refresh Token 換取新 Access Token（Token Rotation） |
| `POST /api/auth/logout` | 使 Refresh Token 失效 → 204 |
| `GET /api/users/me` | 查詢自己的個人資料（需 Bearer token） |
| `PUT /api/users/me` | 更新個人資料（username、顯示名稱） |

**資料庫 Schema**: `member`，Tables: `users`, `refresh_tokens`

### 技術注意事項

- 密碼使用 **BCrypt** 雜湊（`BCrypt.Net-Next`）
- JWT: HS256，Secret 來自 `JWT_SECRET` 環境變數（≥ 32 chars）
- Refresh Token: 儲存在 DB，Rotation 策略（使用後舊 token 失效）
