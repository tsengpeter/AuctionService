## 架構重點備忘（供後續模組參考）

| 事項 | 說明 |
|------|------|
| EF Core 欄位命名 | 無 `UseSnakeCaseNamingConvention` → 欄位為 **PascalCase**（`"Id"`, `"Name"` 等） |
| FluentValidation 注冊 | 每個模組的 DI **必須獨立**呼叫 `services.AddValidatorsFromAssembly(...)` |
| 204 端點 | HTTP 204 禁止帶 response body，直接 `return NoContent()`（不包 `ApiResponse`） |
| Background Service 注冊 | `AddHostedService<T>()` 在 Application DI；integration test 透過 `GetServices<IHostedService>().OfType<T>()` 取得 |
| JWT userId 取得 | `User.FindFirst(ClaimTypes.NameIdentifier)?.Value`；**不信任 request body 中的 userId** |
| 跨模組依賴 | 邏輯 Guid 參考，不建 DB Foreign Key；透過 Interface（如 `IBiddingQueryService`）隔離 |
